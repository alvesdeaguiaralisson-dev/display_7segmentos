# 🔢 Display de 7 Segmentos com Arduino

## 📌 Descrição
Projeto desenvolvido para controle de um display de 7 segmentos utilizando Arduino, com o objetivo de exibir números de 0 a 9 de forma sequencial. O sistema utiliza lógica programada por meio de uma matriz para acionar corretamente cada segmento do display.

## ⚙️ Tecnologias utilizadas
- Arduino
- Display de 7 segmentos
- Linguagem C/C++ (Arduino)

## 🚀 Funcionamento
O projeto utiliza um array bidimensional para representar os estados de cada segmento (a–g) para os números de 0 a 9. A cada ciclo, o Arduino percorre esse array e ativa os pinos correspondentes, exibindo os números em sequência no display com um intervalo de 1 segundo.

## 💡 Lógica aplicada
Foi utilizada lógica de programação com arrays e estruturas de repetição para controlar os segmentos do display de forma organizada e eficiente, evitando código repetitivo.
